Welcome to Timestamp Microservice

Users stories

  1. I can pass a string as a parameter, and it will check to see whether

    that string contains either a unix timestamp or a natural language

    date (example: January 1, 2016).

  2. If it does, it returns both the Unix timestamp and the natural language

    form of that date.

  3. If it does not contain a date or Unix timestamp, it returns null for

    those properties.

Example input:

https://timestamp-microservice.herokuapp.com/March%2019%201994

https://timestamp-microservice.herokuapp.com/764060400

Example output:

{ "unix": 764035200, "natural": "March 19, 1994" }
